## 📱 Application Overview

### System Type
Multi-tenant, role-based healthcare management system for hospitals and clinics

### Core Features
- 👨‍⚕️ **Doctor Prescription Management** - Fast, keyboard-friendly prescription creation
- 🏥 **Multi-Hospital Support** - Complete data isolation with hospital-specific access
- 👥 **6 Role-Based Access Levels** - Super Admin, Admin, Doctor, Receptionist, Pharmacist, Lab Technician
- 🧪 **Laboratory Management** - Test ordering and result entry workflow
- 💊 **Pharmacy Management** - Medicine inventory and dispensing
- 📅 **Appointment Scheduling** - Patient appointment management
- 🔐 **Enterprise Security** - Role-based permissions and data isolation

### Technology Stack
- **Frontend**: React 18 + TypeScript + Tailwind CSS v4
- **State Management**: React Context API
- **Routing**: Client-side navigation
- **UI Components**: Radix UI + Custom components
- **Charts**: Recharts
- **Internationalization**: i18next (4 languages)
- **Planned Backend**: Laravel 12 + MySQL
